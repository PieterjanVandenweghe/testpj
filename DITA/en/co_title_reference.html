Titling the reference information: <title>

To add a title to a reference topic, you use the <title> element. It is best to use noun phrases in reference titles, so users can easily distinguish them from task titles (which use verb phrases or how-to phrases). Distinguishing reference titles from concept titles is harder, because these titles are noun-based as well. However, you can still give users hints about the type of information the topic includes.

For example, you are writing a reference topic that lists all DITA XML elements. If you title your topic "DITA elements", that topic could also describe what DITA XML elements are (concept topic). You can solve this problem by using more specific titles, such as "What are DITA XML elements?" for the concept topic and "Overview of DITA XML elements" for the reference topic.
