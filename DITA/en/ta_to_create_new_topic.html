To create a new topic

  1. Choose File > New.
  2. Go to Framework templates > DITA > topic.
  3. Select the information type of the topic that you want to create and click Create.
    Note: Keep in mind the Guidelines for file naming.
    Oxygen opens a .dita file of the selected information type. It already contains some elements that are specific for that information type.
