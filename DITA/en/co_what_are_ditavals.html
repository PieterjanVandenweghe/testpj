What are ditavals?

DITAVALs are used to define conditional processing. There are different types of processing, which will be outlined here.

Exclude

If you want to filter content, you add strings of text to the DITAVAL file, for example: <prop att="audience" val="expert" action="exclude">. During publication, this element will be excluded in the output.

Flag

By flagging content, you highlight certain elements with a graphic style or give them icons to represent that they only apply for, for example, one appliance model/software version/operating system. These attributes are also logged in a DITAVAL file where you specify a flag to highlight the content. You can do this by:
  • Setting the action to flag instead of exclude.
  • Using the <startflag> element to insert an image to highlight the conditional content.
  • Using the <alt-text> element to provide alternative text for the image.
You create a DITAVAL file and you specify strings for each Word value:
<val>
<prop att="platform" val="2003" action="flag">
<startflag imageref="Word_2003_Icon.jpg">
</startflag>
</prop>
<prop att="platform" val="2007" action="flag">
<startflag imageref="Word_2007_Icon.jpg">
</startflag>
</prop>
</val>
