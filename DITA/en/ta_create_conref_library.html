To create a conref library

  1. Choose File > New.
  2. Go to Framework Templates > DITA > Topic.
  3. Choose 'Composite' and click Create.
    A composite file contains all topic formats in one, allowing you to add elements that can only be found in specific topic types to one file.
  4. Choose File > Save and name your file !conref_library.
    Tip: Adding an exclamation mark to your filename ensures the file appears at the top of your list of files in your folder, making it easier to find.
  5. Fill this library while you write by immediately adding elements you intend to re-use in other topics to this conref library.
  6. Give each element an ID by right-clicking it and choosing Generate IDs.
