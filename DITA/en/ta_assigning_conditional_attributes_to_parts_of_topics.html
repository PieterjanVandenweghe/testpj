Assign conditions to a part of a topic

  1. In Oxygen, select the element you want to apply a conditional attribute to.
  2. In the Attributes list, click in the empty field next to a conditional processing attribute and enter a value.
  3. Press Enter.
  4. In Oxygen, you can see which elements you have applied an attribute to by choosing DITA > Profiling/Conditional Text > Show profiling attributes.
