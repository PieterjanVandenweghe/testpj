What is renaming elements?

Renaming an element is changing that element into another element. This can be useful if you accidentally wrapped your text in incorrect tags or if you are reorganizing a topic.

For example, to promote all the paragraphs in a particular topic to sections, rename all the <p> elements to <section> elements.
