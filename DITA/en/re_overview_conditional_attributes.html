Overview of conditional attributes

Attribute Purpose
@audience Used for the intended audience of the information. For example: novice, advanced, administrator, Canada, France...
@platform Used for operating systems or similar environments. For example: Windows, Linux, Mac, Firefox, mobile, Android...
@product Used for a product or component value. For example: RayGun 2000, RayGun N81....
@rev Used to record the revision level of a particular topic. You can use this attribute only to highlight content, not to filter content.
@props A generic attribute that is used to specialize new conditional processing attributes.
@otherprops A miscellaneous attribute with no specific semantic meaning. You can use this attribute for values that don't fit in the other default attributes. For example: market, publication, organizational department, outlet...
