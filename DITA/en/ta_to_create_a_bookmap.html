To create a new bookmap

  1. Choose File > New.
  2. Go to Framework templates > DITA Map > map.
  3. Choose Bookmap and click Create.
    In the DITA Maps Manager, Oxygen opens an untitled .ditamap file of the type that you selected.
  4. Click File > Save and give the map a name.
    Tip: As Oxygen does not make a visible distinction between bookmaps and regular DITA maps, it helps to add _bookmap to the filename. For example, filename_bookmap.ditamap.
