To insert a table using the table wizard

Instead of inserting a <table> or <simpletable> element and creating your table from scratch, you can use the table wizard to insert a table in your topic. Proceed as follows:
  1. Place you cursor where you want to insert the table and do one of the following:
    • Click in the Main toolbar.
    • Press Enter and click table.
  2. In the Insert table dialog, define all the properties of your table:
    Property Description
    Model Select CALS for a common table and Simple for a simple table
    Title Check this box if you want to add a title to your table. This is not possible for simple tables.
    Table size Indicate how many rows and columns your table has to include. You can always add and remove rows or columns later.
    Generate table header Check this box if you want to include a header row in your table.
    Column widths Select how the column widths are determined:
    • proportional: The width of columns is specified in relative units of measure. The value of the shares is totaled and rendered as a percent, for example 16.7%, 33.3%, and 66.7%. If you change the width by dragging a column in Author mode, the values are automatically changed accordingly. By default, when you insert, drag and drop, or copy/paste a column, the value is 1*.
    • dynamic: If you choose this option, the columns are created without a specified width. Entering content into a cell changes the rendered width dynamically. If you change the width by dragging a column in Author mode, a dialog box will be displayed that asks you if you want to switch to proportional or fixed column widths.
    • fixed: Columns will be displayed in the width you manually assign to them. This is the least suitable option if you are working with "responsive" output.
    Frame Select where a border should appear in the table. The allowed values are as follows:
    • none: No border will be added.
    • all: A border will be added to all frames.
    • top: A border will be added to the top frame.
    • topbot: A border will be added to the top and bottom frames.
    • bottom: A border will be added to the bottom frame.
    • sides: A border will be added to the side frames.
    Row separator
    • 0: No visible separators
    • 1: Visible separators
    Column separator
    • 0: No visible separators
    • 1: Visible separators
    Alignment Select how to align the table in the output.
  3. Click OK.
