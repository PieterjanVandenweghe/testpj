To add metadata to a DITA map

  1. Open your DITA map in your Editor area.
  2. Underneath your <title> element, but still within the <map> elements, add the element <topicmeta>.
  3. Within <<topicmeta>, add an element <metadata>.
    Tip: While metadata elements are also directly available in the <topicmeta> element, it is recommended to use the ones available in the <metadata> element, as it better supports re-use between topics and maps.
    Note: Metadata in maps will override metadata found in topics, unless otherwise specified.
  4. Add the metadata elements you require.
