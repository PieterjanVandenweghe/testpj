Main components of a bookmap

Component list

Bookmaps contain components which are similar to the parts of a traditional book. The topic references therefore are labeled according to the book components they represent. A bookmap can contain:
Component Purpose
Booktitle Contains the title information for a book, including the library title, main title, subtitle, and other titles.
Bookmeta Contains copyright information, author information, publishing data.
Frontmatter Contains cover, table of contents.
Chapter Groups topic references.
Part Divides chapters into logical groups.
Appendix Contains notices, dedications.
Backmatter Contains legal notices, glossaries.
Relationship table Defines relationships between topics.
Attention: A bookmap should contain no content, but also no direct topic references. Instead, bookmaps should contain only DITA map references which, in turn, can contain topic references. These DITA maps are then the chapters or parts of your book.
