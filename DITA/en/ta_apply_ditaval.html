To apply a ditaval during publishing

These instructions show how to apply a DITAVAL using the built-in Open Toolkit transformation options of Oxygen. If you are using a DITA2PDF system, a different method should be used.
  1. Click Configure Transformation Scenario(s) .
    • If you want to publish a single file, click this button in the Main toolbar.
    • If you want to publish your entire DITA map, click this button at the top of your DITA Maps Manager.
  2. In the Configure Transformation Scenario(s) window, select the transformation scenario that best fits your needs and click Duplicate.
  3. In the Edit DITA scenario window, do the following:
    1. Enter a unique name for your new scenario, so you can easily remember which DITAVAL file is associated with it.
    2. In the Filters tab, select Use DITAVAL file.
    3. Browse to the correct file and click OK.
  4. Click OK.
    Your new transformation scenario appears in the second list.
  5. Select and deselect the necessary transformation scenarios.
    Note: The original transformation scenario will automatically remain selected. Don't forget to deselect it if you only want to apply your newly created scenario.
  6. Click Apply associated to run the transformation scenario you just created.
    Oxygen now publishes your document. Depending on the size of your documentation, this may take a while.
