Output formats of the DITA OT

These are some of the most common standard output formats for the DITA OT (as supported by Oxygen).

PDF

Adobe Portable Document Format (PDF) is a compact binary file format that can be viewed and printed by anyone, anywhere across a broad range of hardware and software using the free PDF Viewer from Adobe. For several years, it has been the standard format used to exchange documents that need to be reproduced independently from their application software, hardware or operating system.

WebHelp

WebHelp or HTML Help is the standard help system for the Windows platform. Webhelp allows you to create online help for a software application or to create content for a multimedia title or website. Oxygen offers two additional features to this:

  • WebHelp - Mobile: documentation for mobile devices such as phones and tablets. It is compatible with all major mobile operator systems.
  • WebHelp with Feedback: includes an option for the readers to post and view comments and questions about the topic. This feature allows users to communicate directly with each other and with the creators.

XHTML

XHTML stands for eXtensible HyperText Markup Language and thus is related to XML markup languages. It is similar to HTML, the language of webpages, but while almost identical to HTML 4.01, XHTML is a stricter and cleaner version of HTML. XHTML is HTML defined as an XML application.

EPUB

EPUB (electronic publication) is a free, open e-book standard. EPUB output is suited for e-book devices of all sorts, meaning the text can be customized by the reader to suit their device.

CHM

CHM is the extension of Microsoft Compiled HTML Help. It is an online help format that contains HTML pages, a hyperlinked table of contents, an index and a search function to navigate through the information.

ODF

ODF, also known as OpenDocument, stands for Open Document Format for Office Applications. It is XML based and is intended for documents containing text, spreadsheets, charts, and graphical elements. The file format makes transformations to other formats simple by leveraging and reusing existing standards wherever possible. For example, the Oxygen XML Editor offers support for editing, transforming and validating documents composing the Open Document format (ODF) package directly through the archive support.

RTF

RTF stands for Rich Text Format and was developed by Microsoft for cross-platform document interchange. It is a format most word processors are able to read and write.

Eclipse Help

Eclipse Help is the help system incorporated in the Eclipse platform that enables Eclipse plugin developers to incorporate online help in their plugins. Eclipse Help contains XHTML output, CSS files and some control files.
