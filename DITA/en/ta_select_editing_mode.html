To select an editing mode

Once you have created a DITA topic in oXygen, you can look at and edit that topic in different editing modes. Each editing mode shows the topic in a specific way:
Editing mode Description
Text editing mode Shows the XML source of the topic.
Grid editing mode Shows the topic as a structured grid of nested tables.
Author editing mode Shows the topic in a WYSIWYG like editor.

To edit topics, it is easiest to use the Author editing mode. It can be interesting, however, to check how a particular element looks in the XML source of your topic with the Text editing mode. To select an editing mode:

Click the Text, Grid or Author button at the bottom of the Editor area.
