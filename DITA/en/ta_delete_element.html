To delete an element

CAUTION: When you delete an element, you also delete the content (text, table, illustration...) inside that element. If you only want to remove the element tags from a piece of content, you have to unwrap the element.
  1. Select the element that you want to delete by clicking one of the surrounding element tags.
  2. Press Delete or Backspace.
