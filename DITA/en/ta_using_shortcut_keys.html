Using shortcut keys

Actions that you use regulalry can be made easier to do with shortcut keys.

  1. Go to Options > Menu Shortcut Keys....
  2. In the Search bar, look for a feature, for example Rename element.
  3. Click the feature in the list and click Edit.
  4. Press the key combination you want to use.
    The shortcut will appear in the field.
  5. Click OK.
    The shortcut is added.
  6. Select Global Options or Project Options to choose whether you want to use the shortcut in all projects or only this one.
  7. Click Apply and OK.
