To add a table of contents to a bookmap

  1. Open your bookmap in your Editor area.
  2. Right-click the bookmap and select Append child > Frontmatter
  3. Within <frontmatter>, add an element <booklists>.
  4. Within <booklists>, add an element <toc>.

Oxygen will create a placeholder in which the table of contents will be generated once you create output for your files.
