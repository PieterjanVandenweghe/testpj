To rename an element

  1. Right-click one of the element tags that surrounds the element you want to rename.
  2. Choose Refactoring > Rename Element.
  3. Enter the new name of the element or select it in the drop-down list.
  4. Indicate if you want to rename the selected element, all siblings with the same name or all elements with the same name.
    For example, to promote all paragraphs in your topic to sections, select Rename siblings with the same name or Rename all elements with the same name.
  5. Click Ok.
