To create a new ditamap

  1. Choose File > New.
  2. Go to Framework templates > DITA Map > map.
  3. Choose the type of map you to would like to create and click Create.
    In the DITA Maps Manager, Oxygen opens an untitled .ditamap file of the type that you selected in the DITA Maps Manager.
  4. Click File > Save and give the map a name.
